# Gauteng-Rooms

Welcome to **Gauteng-Rooms**! This is a simple website project that lists available rooms and rental properties in Gauteng, South Africa. The project is designed for beginners learning web development and GitHub deployment.

---

## 📂 Project Contents

- `index.html` – The main webpage of the Gauteng-Rooms site.
- `style.css` – Optional styling for the webpage.
- `script.js` – Optional JavaScript functionality.
- `README.md` – This file, explaining the project.

---

## 🌐 About the Website

**Gauteng-Rooms** is a website to showcase rooms, rental properties, or shared accommodation available in different areas of Gauteng, including:

- Johannesburg
- Midrand
- Sandton
- Pretoria
- Centurion
- Soweto
- And more…

The goal of the website is to provide a simple, easy-to-navigate interface for anyone searching for rental options.
